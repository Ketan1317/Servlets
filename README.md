<h1>Java Servlets Web Application 🚀</h1>

<p>
This repository demonstrates the implementation of <b>Java Servlets</b> and <b>JSP (Java Server Pages)</b>
for building dynamic web applications using the Java EE architecture.
</p>

<p>
The project shows how servlets handle HTTP requests and responses,
while JSP pages are used to create dynamic web interfaces.
</p>

<hr>

<h2>▶️ How to Run the Project</h2>

<h3>1️⃣ Clone the Repository</h3>

<pre>
git clone https://github.com/Ketan1317/Servlets.git
</pre>

<h3>2️⃣ Open the Project</h3>

<p>
Import the project into an IDE such as IntelliJ IDEA, Eclipse, or VS Code.
</p>

<h3>3️⃣ Configure the Server</h3>

<p>
Set up <b>Apache Tomcat</b> as the application server inside your IDE.
</p>

<h3>4️⃣ Deploy the Project</h3>

<p>
Deploy the project to the Tomcat server.
</p>

<h3>5️⃣ Run the Application</h3>

<p>
Open your browser and navigate to:
</p>

<pre>
http://localhost:8080/Servlets
</pre>
